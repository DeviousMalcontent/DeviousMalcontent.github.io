Binary Numbering System

Binary is a base-2 numeral system, which represents numeric values using two different symbols: typically 0 (zero) and 1 (one).

Base: 2, valid digits: 1 0, weighting of successive digits: 20 21 22 23 24 25 26 27

To convert a Binary number into Decimal form consider the following table:

27 26 25 24 23 22 21 20
128 64 32 16 8 4 2 1
1 0 0 1 1 0 1 1

For each radix which has a value of '1' inputted that position will need to be summed, for instance:
128 + 16 + 8 + 2 + 1 = 155, the decimal value is of 10011011 is 155.
