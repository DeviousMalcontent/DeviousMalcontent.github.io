Decimal Numbering System

The decimal numeral system (also called base ten or occasionally denary) has ten as its base. It is the numerical base most widely used by modern civilisations.

Base: 10, valid digits: 1 2 3 4 5 6 7 8 9 0, weighting of successive digits: 1's, 10's, 100's and 1000's

To convert Decimal to Binary you continuously divide the decimal number by the Binary radix of 2 then save the remainder.

For instance the decimal number 235:

235 / 2 = 117.5 1 ───────────┐
117 / 2 = 58.5 1 ──────────┐│
58 / 2 = 29 0 ─────────┐││
29 / 2 = 14.5 1 ────────┐│││
14 / 2 = 7 0 ───────┐││││
7 / 2 = 3.5 1 ──────┐│││││
3 / 2 = 1.5 1 ─────┐││││││
1 / 2 = 0.5 1 ────┐│││││││
0 / 2 = 0 0 ───011101011
