Octal Numbering System

The octal numeral system is the base-8 number system, and uses the digits 0 to 7.

To convert Decimal to Octal you continuously divide the decimal number by the Octal radix of 8 then save the first digit in the remainder, until the Quotient reaches zero.

For instance the decimal number 984:

Number Quotient Remainder Octal Result
984 / 8 = 123 0 0
123 / 8 = 15 3 30
15 / 8 = 1 7 730
1 / 8 = 0 1 1730
